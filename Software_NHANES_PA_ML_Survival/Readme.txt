=============================
TITLE: Contains code and dataset from the paper "Daily Physical Activity Patterns Enhance Mortality Prediction in NHANES 2011-2014: A Comparative Study across Classical, Machine Learning, and Deep Learning Models using Accelerometer Data".
AUTHORS: Srimanta Ghosh, Sunwoo Emma Cho, Yuanzhen Yue, Marcos Matabuena, Enakshi Saha, Rahul Ghosal.
EMAIL: rghosal@mailbox.sc.edu (contact for any queries)
==============================


1. Folder "Code" contains all necessary code to replicate the results in the paper. 
a) Within subfolder R "Cox + RSF. R" contains all code for implementing the Cox and RSF model in R. "NHANES_11_14_survPA_TD_Final.RData"
is the main data file for analysis. Other codes for additional analysis and performance evaluation are also included.

 
b) Within subfolder Python "DeepSurv" and "GBSM_Fpc" contains all main code for implementing the Deepsurv and GBSM model in Python. Other codes for additional analysis and performance evaluation are also included.

 
2. Folder "Dataset" contains the main dataset for analysis "NHANES_11_14_survPA_TD_Final.RData" and all produced output files in the analysis.
